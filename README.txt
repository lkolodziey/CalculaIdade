Console application desenvolvida em .NET 5 portanto, para compilar precisa ter o SDK desta versão instalada.
A versão utilizada do Visual Studio, no desenvolvimento desta aplicação foi: Microsoft Visual Studio Enterprise 2019 Version 16.8.2
Versões anteriores a 16.8.x não dão suporte para o .NET 5, portando é obrigatório atualizar a plataforma.
